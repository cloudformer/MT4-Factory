## 位置说明

### Toast通知位置
- 📍 **右上角** (top-4 right-4)
- 📏 最小宽度 320px (min-w-80)
- 📏 最大宽度 448px (max-w-md)
- 🎭 滑入动画（从右侧滑入）
- ⏱️ 自动消失（4秒后）

### 状态横幅位置
- 📍 **账户管理Tab顶部**
- 📏 全宽
- 🎨 圆角 + 边框 + 半透明背景

## 颜色方案

### macOS Mock模式
```css
/* 横幅 */
background: rgb(30 58 138 / 0.2);  /* bg-blue-900/20 */
border: #1d4ed8;                    /* border-blue-700 */

/* 文字 */
标题: #60a5fa;                      /* text-blue-400 */
正文: #d1d5db;                      /* text-gray-300 */
说明: #9ca3af;                      /* text-gray-400 */
```

### Windows成功模式
```css
/* 横幅 */
background: rgb(20 83 45 / 0.2);   /* bg-green-900/20 */
border: #15803d;                    /* border-green-700 */

/* 文字 */
标题: #4ade80;                      /* text-green-400 */
```

### Windows失败模式
```css
/* 横幅（严重错误）*/
background: rgb(127 29 29 / 0.2);  /* bg-red-900/20 */
border: #b91c1c;                    /* border-red-700 */
标题: #f87171;                      /* text-red-400 */

/* 横幅（警告）*/
background: rgb(113 63 18 / 0.2);  /* bg-yellow-900/20 */
border: #a16207;                    /* border-yellow-700 */
标题: #fbbf24;                      /* text-yellow-400 */
```

## 实际查看步骤

1. **打开浏览器**: http://localhost:8001
2. **点击Tab**: "账户管理"
3. **观察顶部**: 蓝色Mock模式提示框
4. **点击账户**: 展开查看详细信息
5. **点击按钮**: 触发Toast通知

## 控制台提示

打开浏览器控制台（F12），可以看到：

```
⚠️ 当前为Mock模式（macOS/Linux），显示模拟数据
   提示：真实MT5数据需要在Windows系统上运行
```

## 与Windows对比

| 特性 | macOS | Windows (成功) | Windows (失败) |
|------|-------|---------------|---------------|
| **横幅颜色** | 🔵 蓝色 | 🟢 绿色 | 🔴 红色/🟡 黄色 |
| **图标** | ℹ️ | ✅ | ❌/⚠️ |
| **标题** | Mock模式（开发环境） | MT5连接成功 | MT5未安装/连接失败 |
| **内容** | 显示模拟数据说明 | 余额+净值+同步时间 | 错误详情+解决方案 |
| **提示性质** | 信息提示 | 成功提示 | 错误提示 |
| **数据来源** | 配置文件静态数据 | MT5实时数据 | 数据库缓存 |

## 用户体验

### macOS开发者
✅ 清楚知道当前是Mock模式  
✅ 了解数据是模拟的  
✅ 知道需要Windows才能看真实数据  
✅ 不会误以为是真实MT5数据

### Windows用户
✅ 一眼看出连接状态（绿色/红色/黄色）  
✅ 知道数据是否实时同步  
✅ 遇到问题有明确解决方案  
✅ 余额和同步时间一目了然
